Module 3: Gaming

Step 1: First Steam Games

⚡ Fast track

Open Steam from the bottom menu or applications.

First login:
  1. Steam will update itself (wait a minute)
  2. Log in with your Steam account
  3. Go to Library
🎮 Pick a game and click "Install"

Games install just like on Windows.

🐢 I want to understand

Steam on Bazzite – what's different?

  • Proton: Bazzite includes Proton, which makes Windows games run on Linux
  • Most games work: Check ProtonDB for your favorites
  • Performance: Often same or better than Windows!

Settings to check:

  • Steam → Settings → Compatibility → Enable Steam Play for all titles
  • Use Proton Experimental for best compatibility

Non-Steam games:

Heroic Launcher (Epic/GOG) and Lutris (other stores) – we'll cover later.

✅ Checkpoint

Did a game install and run?

Common issues:

  • Game won't start? Check ProtonDB – maybe need a specific Proton version
  • No sound? Check Steam audio settings
  • Controller not working? Bazzite supports Xbox/PS controllers natively
  • Still stuck? Ask Bazzi at Campfire
