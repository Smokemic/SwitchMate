Module 4: Emergency Kit

Step 1: Boot Problems

⚡ Fast track

If Bazzite won't boot:

  1. Restart and hold Shift (or spam Esc)
  2. You'll see a boot menu
  3. Choose "Advanced options"
  4. Select an older kernel (version)
💾 Boot with previous kernel
🐢 I want to understand

Why won't it boot?

  • Bad update: Sometimes a new kernel has issues
  • Graphics driver: NVIDIA/AMD driver problems
  • Corrupted files: Rare, but possible

Solutions:

  • Boot old kernel: Safe option, works 90% of time
  • Boot from USB: Use your installation USB to repair
  • Rollback update: rpm-ostree rollback in terminal
Remember: Your Windows is still there! Hold F2/Del at boot to choose Windows.

✅ Checkpoint

Did you manage to boot Bazzite (even with old kernel)?

Last resort options:

  • Boot from USB: Use your installation USB, choose "Try Bazzite"
  • Mount and backup files: From USB, you can access your disk
  • Reinstall: From USB, you can reinstall (keeps your files if you choose wisely)
  • Windows: Boot to Windows via BIOS (F2/Del) – everything is still there
