Module 4: Emergency Kit

Step 3: Going Back to Windows

⚡ Fast track

If you want to remove Bazzite and keep only Windows:

  1. Boot Windows normally (choose in boot menu)
  2. Open Disk Management (right-click Start)
  3. Find Bazzite partitions (EXT4, no drive letter)
  4. Delete them, extend Windows partition
⚠️ This will delete all Bazzite files!

Or use command line (admin):

diskpart → list disk → select disk → list partition
🐢 I want to understand

What happens to Bazzite files?

If you delete the partitions:

  • ❌ All Bazzite files, apps, settings are gone
  • ❌ Games installed on Linux partitions are gone
  • ✅ Files on shared NTFS (Windows) partitions stay

Alternative: Keep dual boot

You can just keep both! At boot, choose:

  • Windows Boot Manager → Windows
  • Bazzite → Bazzite

If boot menu disappeared:

Enter BIOS (F2/Del) and change boot order back to Windows Boot Manager.

Pro tip: Backup important files from Bazzite first! Boot from USB, mount disk, copy files.

✅ Checkpoint

Are you back to Windows only? Or happy with dual boot?

Windows boot problems:

  • No boot menu: Enter BIOS (F2/Del), set Windows Boot Manager as first
  • Windows doesn't start: Use Windows installation USB → Repair
  • Bootrec commands: bootrec /fixmbr, bootrec /fixboot
  • Grub still there: You can keep it – just choose Windows
