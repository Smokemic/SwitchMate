Module 4: Emergency Kit

Step 2: Fix Broken Packages

⚡ Fast track

If apps won't start or system acts weird:

ujust update

If that doesn't help:

rpm-ostree status
rpm-ostree rollback
Rollback = go back to previous working version
🐢 I want to understand

How Bazzite updates work:

Bazzite uses rpm-ostree – it's like a "git for your system". Each update is a new snapshot.

Benefits:

  • ✅ You can always rollback to previous version
  • ✅ Updates are atomic (all or nothing)
  • ✅ System is read-only – harder to break

Common fix commands:

  • ujust update – Normal update
  • rpm-ostree rollback – Go back one version
  • rpm-ostree status – See available versions
  • ujust repair – Try to fix common issues
Pro tip: After rollback, you can pin the working version with rpm-ostree pin 0

✅ Checkpoint

Is your system working again? Apps starting?

Last resort:

  • Boot from USB: Use your installation USB
  • Backup files: Copy important stuff to external drive
  • Reinstall: From USB, choose "Reinstall Bazzite" (keeps /home)
  • Complete reset: Fresh install (backup first!)
