Module 1: The USB Stick

Step 1: Boot from USB

⚡ Fast track

Restart your PC with the USB stick plugged in.

Press the boot menu key during startup:

F12, F8, or Esc

Then select your USB stick from the list.

→ USB drive / UEFI: USB
🐢 I want to understand

What's happening?

You're telling your PC: "Don't start Windows – start from the USB stick instead."

Common boot keys by brand:

  • HP: F9 or Esc
  • Dell: F12
  • Lenovo: F12 or Novo button
  • Acer: F12
  • Asus: F8 or Esc
  • MSI: F11
Pro tip: If the boot menu doesn't work, enter BIOS (F2/Del) and change boot order there.

✅ Checkpoint

Did you see a menu with your USB stick? Did you select it?

Common issues:

  • USB not in boot menu? Try a different USB port (USB 2.0 often works best)
  • Still Windows starts? Disable Fast Boot in BIOS
  • Secure Boot blocking? Go back to Module 0 Step 2
  • Nothing helps? Ask Bazzi at Campfire
