Module 1: The USB Stick

Step 3: Install Bazzite

⚡ Fast track

On the Bazzite desktop, double-click:

💿 Install Bazzite
  1. Select language → English
  2. Select drive → Your internal SSD/HDD
  3. Click "Install"
  4. Wait 5-10 minutes
🐢 I want to understand

What happens during installation?

  • Bazzite is copied from USB to your internal drive
  • It creates a dual-boot menu (Windows + Bazzite)
  • Your Windows files stay untouched

Important choices:

  • Drive: Pick the right one! Usually the largest internal drive
  • Partitioning: Choose "Use free space" or "Install alongside Windows"
  • User account: Create a username and password
Warning: Don't select "Erase entire drive" unless you're sure!

✅ Checkpoint

Do you see "Installation Complete"?

Common issues:

  • Can't find drive? Check BIOS: SATA mode = AHCI
  • Installation freezes? Try "Basic Graphics" mode from boot menu
  • Windows not detected? You can still install, but dual-boot must be set up manually
  • Still stuck? Ask Bazzi at Campfire
