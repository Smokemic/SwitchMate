Module 2: First Days

Step 2: First Updates

⚡ Fast track

Open a terminal (right-click desktop → Open Terminal) and run:

ujust update
This will:
  • Update Bazzite to the latest version
  • Install any new drivers
  • Take 5-15 minutes

When asked for password, type your login password.

🐢 I want to understand

Why update immediately?

Your USB stick might have an older version. Updating ensures:

  • ✅ Latest security fixes
  • ✅ Newest drivers (especially important for NVIDIA)
  • ✅ Bug fixes and improvements

What is "ujust"?

Bazzite has special helper commands. ujust update is the simplest way to update everything.

Alternative:

Click the update icon in the bottom right (if visible). But terminal is more reliable.

✅ Checkpoint

Do you see "System updated successfully"?

Common issues:

  • Password not working? Type carefully – no visual feedback in terminal
  • Update too slow? Check your internet connection
  • Error messages? Take a photo and ask at Campfire
  • Command not found? Try: rpm-ostree update
