Module 2: First Days

Step 1: First Boot & Setup

⚡ Fast track

After installation, remove USB and restart.

You'll see a boot menu:

Choose "Bazzite"

Bazzite will start. Then:

→ First Setup Wizard
  1. Language → English
  2. Keyboard → English (US)
  3. WiFi → Connect
  4. Location → Your city
  5. Create user → Your name, password
🐢 I want to understand

What's happening?

This is the first real boot from your internal drive. The setup wizard configures:

  • User account: This will be your main login
  • Password: Also used for admin (sudo) tasks
  • Automatic login: Optional – you can change later

Important:

  • Write down your username and password!
  • If you have an NVIDIA GPU, drivers will install automatically after first login
Pro tip: Use a simple password for now – you can change it later.

✅ Checkpoint

Are you looking at the Bazzite desktop? Logged in?

Common issues:

  • Still boots to Windows? Change boot order in BIOS (F2/Del)
  • Bazzite not in boot menu? Installation might have failed – try Module 1 again
  • NVIDIA black screen? Boot in "Basic Graphics" mode and install drivers
  • Forgot password? Ask Bazzi at Campfire
