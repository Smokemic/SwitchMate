## 🇬🇧 English

**The idiot-proof switch from Windows to Bazzite.**

Switchmate isn't another "learn Linux" tutorial. Switchmate is your personal companion that guides you step by step through the installation – no jargon, no shortcuts, no "it's easy".

### 🎯 Who is Switchmate for?

- You're done with Windows, but know nothing about Linux
- You want to try Bazzite, but are afraid of breaking something
- You're looking for a guide that holds your hand – not just text
- You just want to game, not spend months learning Linux

### 📱 Website + App – One Codebase

| **Website** | **PWA (App)** |
|-------------|---------------|
| Runs on GitHub Pages | Installable on phone/tablet |
| For browsing on the go | Works offline |
| Same code | Looks like a native app |

**One codebase. Two platforms. Zero extra work.**

### 🧭 The 5 Modules

| Module | Title | What to expect |
|--------|-------|----------------|
| **0** | **Before the Leap** | Check hardware (GPU, Secure Boot), prepare USB stick, backup |
| **1** | **The USB Stick** | Download tool, burn ISO, create bootable stick – with error images! |
| **2** | **Installation** | Boot from USB, click through installer, create user, reboot |
| **3** | **First Days** | Updates, first apps, transfer data, test Steam and gaming |
| **4** | **Emergency Kit** | If something goes wrong: recovery, boot problems, back to Windows |

### 🎨 Design Philosophy

**Clear. Safe. Unmistakable.**

- **Every step has an image.** You see how it SHOULD look – and how it CAN look if something's different.
- **Every command is copyable.** Never type, just click.
- **Two tracks per step:** "Quick to goal" or "I want to understand".
- **Colors with meaning:**
  - 🟦 Blue: Trust, base, "All good here"
  - 🟩 Green: Success, checkpoints, "Keep going"
  - 🟧 Orange: Caution, "Something's about to happen"
  - 🟥 Red: Error – but **always with a solution**

### 🔥 Campfire – The Playground

Switchmate has a heart: **Campfire**.

Here you can:
- Relax by the 3D fire
- Chat with Bazzi your helper 
- Experiment safely in the terminal playground
- Deepen what you've learned without consequences

Campfire is the reward after each completed module. And the place where the curious can understand everything.

### 🛠️ Tech Stack

- **Hosting:** GitHub Pages (free, HTTPS, PWA-ready)
- **App:** PWA – installable, offline-capable
- **Languages:** JSON files for easy expansion
- **Updates:** Push to GitHub → automatic update of website + app

## 🇩🇪 Deutsch

**Der idiotensichere Umstieg von Windows zu Bazzite.**

Switchmate ist kein weiteres "Lern Linux"-Tutorial. Switchmate ist dein persönlicher Begleiter, der dich Schritt für Schritt durch die Installation führt – ohne Fachchinesisch, ohne Abkürzungen, ohne "das ist doch einfach".

### 🎯 Für wen ist Switchmate?

- Du hast Windows satt, aber keine Ahnung von Linux
- Du willst Bazzite ausprobieren, hast aber Angst, etwas kaputt zu machen
- Du suchst einen Guide, der dich an die Hand nimmt – nicht nur Texte liefert
- Du willst einfach nur zocken, nicht monatelang Linux lernen

### 📱 Website + App – eine Codebasis

| **Website** | **PWA (App)** |
|-------------|---------------|
| Läuft auf GitHub Pages | Installierbar auf Handy/Tablet |
| Für unterwegs und zum Schmökern | Funktioniert offline |
| Gleicher Code | Sieht aus wie eine native App |

**Ein Codebase. Zwei Plattformen. Null Extra-Aufwand.**

### 🧭 Die 5 Module

| Modul | Titel | Was dich erwartet |
|-------|-------|-------------------|
| **0** | **Vor dem Sprung** | Hardware checken (GPU, Secure Boot), USB-Stick bereitlegen, Backup machen |
| **1** | **Der USB-Stick** | Tool runterladen, ISO brennen, bootfähigen Stick erstellen – mit Fehlerbildern! |
| **2** | **Installation** | Von USB booten, Installer durchklicken, Benutzer anlegen, Neustart |
| **3** | **Die ersten Tage** | Updates, erste Apps, Daten rüberholen, Steam und Gaming testen |
| **4** | **Notfallkoffer** | Wenn was schiefgeht: Recovery, Boot-Probleme, zurück zu Windows |

### 🎨 Design-Philosophie

**Klar. Sicher. Unmissverständlich.**

- **Jeder Schritt ein Bild.** Du siehst, wie es aussehen SOLL – und wie es aussehen KANN, wenn was anders ist.
- **Jeder Befehl kopierbar.** Nie tippen, nur klicken.
- **Zwei Spuren pro Schritt:** "Schnell ans Ziel" oder "Ich will verstehen".
- **Farben mit Bedeutung:**
  - 🟦 Blau: Vertrauen, Basis, "Hier ist alles gut"
  - 🟩 Grün: Erfolg, Checkpoints, "Weiter geht's"
  - 🟧 Orange: Achtung, "Gleich passiert was"
  - 🟥 Rot: Fehler – aber **immer mit Lösung**

### 🔥 Campfire – Der Spielplatz

Switchmate hat ein Herz: **Campfire**.

Hier kannst du:
- Am 3D-Feuer entspannen
- Mit Bazzi quatschen Dein helfer
- Im Terminal-Spielplatz gefahrlos rumprobieren
- Das Gelernte vertiefen, ohne Konsequenzen

Campfire ist die Belohnung nach jedem geschafften Modul. Und der Ort, wo Neugierige alles verstehen können.

### 🛠️ Technik

- **Hosting:** GitHub Pages (kostenlos, HTTPS, PWA-fähig)
- **App:** PWA – installierbar, offline-fähig
- **Sprachen:** JSON-Dateien für einfache Erweiterung
- **Updates:** Push zu GitHub → automatische Aktualisierung von Website + App
